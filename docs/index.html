Computational Methods for Social Science

Sciences Po Bordeaux

Instructor: Mickael Temporão

Course Overview

This course introduces students to the building blocks of machine learning and artificial intelligence using real world big data and applications in political science, sociology or economics. The course follows a “learn-by-doing” approach and will place emphasis on gaining experience by analyzing real-world data and solving problems using free open source software. The course will allow you to apply your theoretical knowledge and articulate the methods learned through various challenges and projects throughout the course. This course is designed for social science students that are not necessarily familiar with quantitative methods or computer programming.

Goals

  1. Collect, process, visualize, analyze and model data using free and open source software.
  2. Leverage computational methods in a fully reproducible end to end academic research paper.
  3. Build awareness of the ethical considerations related to big data and artificial intelligence.

Pre-requisites

The most important prerequisite is motivation to work hard on possibly unfamiliar material.

Materials & Requirements

Students need to have access to a computer in order to participate, complete activities during each session and accomplish the assignments.

TLDR: Make sure you have done the following before the first session:

  1. Join our Slack channel
  2. Download R
  3. Download RStudio
  4. Download LaTeX
  5. Create a GitHub account

Communication

Before the first session you should receive an email invitation to join our Slack channel.

Use slack for questions regarding the content of the course, ask and answer them on our Slack channel. Being active on Slack and answering your peers’ questions is encouraged and will positively impact your participation grade. Furthermore, if you have a question about the topic, it is likely that someone else had the same question. Posting questions and answers publicly allows us all to learn from each of these questions and answers.

Software

This course uses R, which is a free and open-source programming language primarily used for statistics and data analysis. We also use RStudio, which is an easy-to-use interface to program in R and other languages.

Instructions to install or upgrade R can be found here.

In order to generate high-quality reports we will also use LaTeX.

We will also use git and GitHub for version control. This will allow you to track your changes and help you do reproducible research.

Homework assignments will be distributed and submitted via GitHub Classrooms.

Students have access to a DataCamp classroom. You can use this resource for additional practice.

If this does not yet make sense, don’t worry; we’ll cover it in the course.
